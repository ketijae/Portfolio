This was the second part of the fictitious dance school project- this time we overtook other teams work. Our goal was to keep the main idea of website design and to add more design features to emphasise the function the website is serving. We used tools such as project planning, usability tests, brainstorm, XD mockup and wireframes. Website was coded with HTML and CSS (Mobile first)

process

First we started with website testing to find the flaws in the previous design. Then produced individual mood-boards and tested our target group. Based on on the results, final mood-board and sketches were made. Before coding , I was responsible for XD Prototype production ( To see click on the picture)

For coding part I was assigned the “packages” page. For the solutions I implemented “flexbox” and “overlay” features to achieve semantically correct website.
