In this project we had to produce a website for fictitious dance school. Our goal was to produce a simple website with easily understandable functions. We used tools such as design thinking, brainstorm, wireframes, target group interviews and personas. Website was coded with HTML and CSS

process

Our process started with interviews, gathered inspiration, brainstorms and mood-board testing. Based on that I made a final mood- board to set the overall tone for the website Our group assigned tasks and a page which we had to code individually. I was responsible for the landing page with a video-gif to keep the attention of the user and set the overall feel of the website.
